# 15. Find peak element

A peak element is an element that is strictly greater than its neighbors.

Given a **0-indexed** integer array `nums`, find a peak element, and return its index. If the array contains multiple peaks, return the index to **any of the peaks**.

You may imagine that `nums[-1] = nums[n] = -∞`. In other words, an element is always considered to be strictly greater than a neighbor that is outside the array.

You must write an algorithm that runs in `O(log n)` time.

**Example 1:**

```
Input: nums = [1,2,3,1]
Output: 2
Explanation: 3 is a peak element and your function should return the index number 2.
```

**Example 2:**

```
Input: nums = [1,2,1,3,5,6,4]
Output: 5
Explanation: Your function can return either index number 1 where the peak element is 2, or index number 5 where the peak element is 6.
```

sol :

The linear approach to this problem is straightforward. We iterate through the array and check for an element that is greater than its neighbors. For any index `i`, if `arr[i] > arr[i-1]` and `arr[i] > arr[i+1]`, then it is a peak element. We also handle edge cases separately, such as the first and last elements, since they have only one neighbor. This method guarantees finding a peak because at least one peak always exists in any array. However, this approach takes O(n) time because we may need to scan the entire array.

The optimized approach uses binary search to achieve O(log n) time. Instead of checking every element, we look at the middle element and compare it with its neighbors. If it is greater than both, we have found a peak. If the right neighbor is greater, it means a peak must exist on the right side, so we move `lo` to `mid + 1`. Otherwise, we move `hi` to `mid - 1`, meaning the peak lies on the left side. This works because the slope of the array guarantees that moving in the direction of a larger neighbor will eventually lead to a peak. By repeatedly halving the search space, we efficiently find a peak element.

code:

```python
def peakElement(arr):
    n = len(arr)

    # If there is only one element
    if n == 1:
        return 0

    # Check first element
    if arr[0] > arr[1]:
        return 0

    # Check last element
    if arr[n - 1] > arr[n - 2]:
        return n - 1

    # Binary search space
    lo, hi = 1, n - 2

    while lo <= hi:
        mid = lo + (hi - lo) // 2

        # Check if mid is peak
        if arr[mid] > arr[mid - 1] and arr[mid] > arr[mid + 1]:
            return mid

        # Move right
        if arr[mid] < arr[mid + 1]:
            lo = mid + 1
        else:
            # Move left
            hi = mid - 1

    return 0

# Example
arr = [1, 2, 4, 5, 7, 8, 3]
print(peakElement(arr))
```
